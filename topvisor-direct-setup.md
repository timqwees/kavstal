# ЧАСТЬ 5: РЕКОМЕНДАЦИИ ПО ВЕДЕНИЮ

1. **Поисковые кампании** — чистка минус-слов каждую неделю (б/у, бу, пластик, пвх, аренда, лом)
2. **РСЯ** — обновлять креативы раз в 2 недели (баннеры устаревают)
3. **Topvisor** — отслеживать только топ-30, смысла нет смотреть глубже
4. **Бюджет** — 70% поиск, 15% бренд, 10% РСЯ товарная, 5% РСЯ контекст
5. **Приоритеты по товарам:** арматура > трубы > лист > профнастил > швеллер/балка > нержавейка > газгольдер > остальное
6. **Сезонность:** пик строительного сезона апрель-октябрь — увеличивать бюджет на 40%
7. **Акции коррективы в топвизор:** для отслеживания СЧ-ключей с конверсией >1% — см. метрику
